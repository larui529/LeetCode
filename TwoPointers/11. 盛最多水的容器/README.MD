# 11. 盛最多水的容器

问题描述（中文）：给定一个长度为 n 的正整数数组 height，每个元素表示坐标 (i, height[i]) 上的一根垂直线，找出两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。返回容器能装的最大水量。

解题思路（中文）：
- 使用双指针：left 指向头，right 指向尾。计算当前面积 area = (right - left) * min(height[left], height[right])。
- 将高度较小的一侧指针向内移动，因为移动高度较大的那一侧不可能获得比当前更大的面积（高度受限于较小的一侧）。
- 时间复杂度：O(n)，空间复杂度：O(1)。
